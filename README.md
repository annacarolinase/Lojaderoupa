# Lojaderoupa
Criar um sistema para gerenciamento de uma loja de roupas.

# Funcionalidades
1. O software terá usuários, os usuários terão login e senha.
2. O software terá cadrastro dos usuários(Informações como, CPF, nome completo e número).
3. Cadastro de produtos.
4. Cadastro de categorias (camisetas, calças, vestidos, etc.).
5. Cadastro de tamanhos (PP, P, M, G, GG).
8. Carrinho de compras.
9. Finalização da compra.
